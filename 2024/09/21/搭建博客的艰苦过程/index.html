搭建博客的艰苦过程

标签2024-09-21

#2024/9/21

心血来潮也是来搭了个博客

主要是参照了文档 : 从零搭建 Hexo + Github 博客 - 洛谷专栏 (luogu.com.cn)

前面很顺利，但是在ssh连接github时报错ssh: connect to … github.com port 22: Connection refused发现22端口不可用

然后度娘一番改用443端口 ssh: connect to host github.com port 22: Connection refused_ssh:connect-CSDN博客

然后也是非常的顺利，选用了非常好看的Butterlfy主题，最后上传博客后发现无法加载，仔细排查后发现上传的博客在master的分支里，而默认的是main分支，解决方法只需要在setting->pages的branch将main改成master就可以正常加载了。

接下来就是一系列的美化啦! 参考: Hexo搭建个人博客系列【七】- Butterfly主题使用及美化 - 知乎 (zhihu.com)

最后耗费一个晚上+一个下午也是大功告成啦！

#2024/9/22

突然想起可以把之前高中做的摸鱼小网站缝合上去, 于是打开尘封了一年的文件，大改特改，把所有有关数据库的代码和页面全部删掉了，然后改了改文件的路径就差不多了。

然后在butterfly的_config.yml中将侧面的github链接改成了摸鱼小网站的链接嘻嘻。

在这里就可以快乐的摸鱼啦！

tools: Typorra (Typora使用基础教程_typora代码块-CSDN博客)
